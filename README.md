# Agenda

Aplicacion crud android en la cual puedes leer,actualizar,eliminar y editar 
los contactos

## Features
- Fragments
- Navigation component
- Rom database
- Corutines
- ViewBinding
- MVVM
- Demo https://www.youtube.com/watch?v=Gf-AuousTBk
